# Dataverity

Dataverity is a service for monitoring data quality, validating datasets, and ensuring reproducible data pipelines.

The system helps organizations maintain reliable and transparent data workflows by detecting anomalies, validating business rules, and tracking dataset transformations.

## 🚀 Quick Start

Clone the repository:

```
git clone https://github.com/Hanemiyaaaa/DataVerity
```

Run the project:

```
docker compose up
```

Services will be available at:

```
Frontend: http://localhost:3000
Backend API: http://localhost:8000
Airflow: http://localhost:8080
```
